Data Warehouse & Business Intelligence

In this project, we need to build a data warehouse for a hotel group using MySQL and perform ETL using Alteryx to clean, transform, and blend booking data. Visual dashboards were created to show trends like occupancy rates, booking channels, and cancellations, followed by recommendations given to stakeholders to improve business performance.

Data Warehouse - Star Schema Design

I created a star schema in SQL to organize hotel booking data for analysis. It includes a central fact table of bookings linked to dimensions like guests, agents, dates, and hotels. This structure simplifies queries and supports insights into occupancy trends and booking behavior.

Data Preparation

In this step, I loaded the data into Alteryx and performed data profiling and cleaning to ensure consistency. This included formatting, handling missing values, and standardizing fields so the data aligns with the star schema structure for smooth integration and analysis.

Transformations & Visualizations - Recommendations To Stakeholders

Lastly, transformations were applied to improve the dataset, and visuals were created to highlight key trends. Recommendations were then provided to stakeholders based on the insights uncovered.
