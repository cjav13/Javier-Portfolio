UX Design & Prototyping

This UX design and prototyping project focused on developing an app tailored to a pre-defined user persona. The process included identifying the persona, product storyboarding, and creating low/high fidelity prototypes using Adobe XD. Usability testing was conducted with planned tasks and objectives to gather user feedback, and the System Usability Scale (SUS) was used to evaluate the app’s overall user experience. These insights guide and lead to the final improved design of the app. Click below to view/access the prototype!

View Prototype!

Persona, Storyboard, Low-Fidelity Prototype

First, a user persona was defined to represent the target audience’s needs, goals, and pain points, helping guide design decisions from the user’s perspective. This was followed by a simple product storyboard to visualise the user journey and app flow. A low-fidelity prototype was then developed, using Adobe XD, to quickly test layout ideas and overall structure before refining the design further.

High-Fidelity Prototype

Using Adobe XD, a high-fidelity prototype was developed, which closley resembles how the final product would work in both visuals & interactivity. This version would also be used for usability testing from test users to gatheer feedback and improve overall user experience.

Usability Testing

Moving on to product testing, detailed planning was carried out, which included defining test objectives, creating user tasks, and preparing survey questionnaires. User interactions were observed and recorded, while feedback was collected through surveys and interviews.

The System Usability Scale (SUS) was also used to measure overall user satisfaction. These insights helped identify usability issues, design flaws, and guided improvements for the final prototype, giving a clearer understanding of where the product stands in terms of user experience.

Improved & Finalized High-Fidelity Prototype

Finally, the high-fidelity prototype was refined based on user comments and usability testing insights, including SUS scores and observed interactions. These improvements led to the development of the finalized high-fidelity prototype, ready for handoff or further development before launching the app.
