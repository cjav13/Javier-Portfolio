Jenkins

üîç

Jenkins is our automation bot for the data pipeline. Once tasks are approved, it triggers Control-M jobs, runs shell or Hive scripts, validates files, and even executes Python tasks. If something fails, Jenkins highlights the exact step, making debugging quick and efficient. This ensures smooth, reliable, and hands-free deployments across the entire pipeline.

Bitbucket

üîç

Bitbucket is our secure code repository where we store and manage all SQL scripts. For example, ingestion logic, view creation, and transformation queries. It tracks every change made by developers, allowing us to review, compare, and roll back code when needed. Jenkins pulls the latest scripts directly from Bitbucket to run them in production, ensuring consistency and control across the entire pipeline.

Confluence

üîç

Confluence helps us document everything discussed during the requirement gathering with users and developers. It serves as a central hub where everyone can access project details, decisions, and updates, ensuring the entire team remains aligned. It also supports embedded visuals, links to JIRA tickets, and stores version history for easy auditing and reference.
