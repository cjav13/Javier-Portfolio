Quantitative Analysis

This project applies statistical methods to check assumptions and uncover patterns using real-world data. We tested if hospital admissions at Sengkang General Hospital followed a Poisson distribution and examined the link between gender and drunk driving using data from Oklahoma. These analyses helped us support or reject claims through statistical reasoning.

Chi-Square Goodness-Of-Fit Test

In the first task, monthly hospital admission data from Sengkang General Hospital (2019–2023) is analyzed using excel to test whether the number of admissions followed a Poisson distribution. Chi-Square Goodness-of-Fit is tested at a 5% significance level which helped determine if admissions occurred randomly at a consistent average rate or if other factors influenced the data.

Chi-Square Test Of Independence

In the second task, a sample dataset from Oklahoma was analyzed to explore the relationship between gender and drunk driving behavior. A Chi-Square Test of Independence was used and calculated in excel to determine whether the two variables are associated. By comparing observed and expected frequencies, we assessed Curtis Craig’s claim that the law discriminated based on gender in how it addressed drunk driving behavior.

Multiple Linear Regression Model

For the final task, a multiple linear regression model is used to predict car prices based on features like engine size and horsepower. The dataset was cleaned using Python, with domain-informed transformations applied and categorical variables encoded to prepare the data for modeling.

Model Fitting & Testing

Feature selection was done by utilizing domain knowledge and statistical tools. A correlation matrix was created in Python to identify multicollinearity. ANOVA outputs such as F-statistics, p-values, and degrees of freedom (DOF) were then used to assess the significance of different feature combinations and evaluate model fit.
